# CrossZeroConsole
This tic-tac-toe console game is made with [CrossZeroAPI](https://github.com/melon-code/CrossZeroAPI) using `TTCGameProcessor`.

The game provides options to choose game field size and type of players (computer or player).

Console menu for changing settings and starting/restarting the game is implemented via [ConsoleMenuAPI](https://github.com/melon-code/ConsoleMenuAPI).
